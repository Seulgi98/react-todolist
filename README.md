## Software Deployment ⚙️
1. 페이지 접속
2. 새로운 할 일 추가
- 완료한 일은 todo item 클릭 시 체크 표시
- 삭제하고 싶은 일은 todo item hover시 삭제 표시
  
```sh
- App : Main
- CurrentDate : 실시간 Date 정보
- InputForm : todoList 입력 Form
- TodoHeader :todoList Header 정보(할 일 개수)
- TodoItem : todoList checked, deleted 
- TodoList : todoList
```
